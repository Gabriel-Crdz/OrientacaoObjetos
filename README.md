# ORIENTAÇÃO A OBJETOS I
* ***CURSO:*** TADS IFPR-Foz

* ***SEMESTRE:*** 2º
* ***HORA AULA:*** 80h

#### Tecnologias: Java, VS Code

### Objetivo:
Propiciar ao aluno a compreensão dos principais aspectos do paradigma de
desenvolvimento de software orientado a objetos. Realizar a implementação
destes conceitos em uma linguagem de programação.

### Ementa:
- Paradigma Orientado a Objetos
- Conceitos fundamentais da Orientação a Objetos
- Objeto, Classes, Atributos e Métodos
- Encapsulamento
- Heranças
- Associação (Agregação e Composição)
- Polimorfismo
- Interfaces
- Aplicação dos conceitos da Orientação a Objetos em uma linguagem de programação.
